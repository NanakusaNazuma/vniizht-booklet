# Буклет ВНИИЖТ — QR → листаемый буклет на телефоне

## Короткий ответ

**Да.** После публикации на Vercel (или другом хостинге) **любой человек** с телефоном
и интернетом сможет:

1. Навести камеру на QR-код  
2. Открыть буклет в браузере  
3. Листать страницы свайпом  

Wi‑Fi вашего компьютера **не нужен** — нужен только интернет на телефоне.

---

## Как это работает

```
QR на бумаге  →  https://ваш-сайт.vercel.app/booklet  →  буклет на весь экран
```

- `/` и `/booklet` — сам буклет (то, что открывается после сканирования)
- `/qr` — страница, где скачать QR-картинку для печати

---

## Публикация через GitHub + Vercel (бесплатно)

### Шаг 1. Загрузить проект на GitHub

В терминале VS Code:

```bash
cd D:\WebLogisticVNIIZHT
git init
git add .
git commit -m "Буклет ВНИИЖТ с QR"
```

На [github.com](https://github.com) создайте **новый репозиторий** (без README).
Затем:

```bash
git remote add origin https://github.com/ВАШ_ЛОГИН/vniizht-booklet.git
git branch -M main
git push -u origin main
```

### Шаг 2. Подключить Vercel

1. Зайдите на [vercel.com](https://vercel.com) → войти через **GitHub**
2. **Add New… → Project**
3. Выберите репозиторий `vniizht-booklet`
4. Vercel сам определит **Vite** — ничего менять не нужно:
   - Build: `npm run build`
   - Output: `dist`
5. Нажмите **Deploy**

Через 1–2 минуты получите адрес вида:

`https://vniizht-booklet.vercel.app`

### Шаг 3. Скачать QR для печати

1. Откройте в браузере: `https://vniizht-booklet.vercel.app/qr`
2. Адрес подставится автоматически
3. Нажмите **«Скачать QR для печати»**
4. Вставьте картинку на бумажный буклет

### Шаг 4. Проверить с телефона

Отсканируйте QR камерой — должен открыться буклет и листаться свайпом.

---

## Локальная разработка

```bash
npm install
npm run dev
```

- `http://localhost:5173/` — буклет  
- `http://localhost:5173/qr` — QR для печати  

**С телефона localhost не откроется** — только после публикации на Vercel.

---

## Обновление буклета

Изменили презентацию → замените `public/booklet-slides.pdf` и `public/person.png` →

```bash
git add .
git commit -m "Обновление буклета"
git push
```

Vercel пересоберёт сайт автоматически. QR менять не нужно, если адрес сайта тот же.
